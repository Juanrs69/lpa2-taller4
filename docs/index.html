Sistema de Gestión Musical

Frontend moderno desarrollado con Next.js 16 y TypeScript que consume una API de música con FastAPI. Incluye tests automatizados y documentación completa.

Ver en GitHub Comenzar

Acerca del Proyecto

Este proyecto es la continuación del Taller 3, añadiendo un frontend completo y mejoras en la documentación.

Objetivos

  • Desarrollar un frontend moderno con Next.js 16
  • Implementar TypeScript para mayor robustez
  • Crear tests automatizados completos
  • Documentar exhaustivamente el proyecto

Funcionalidades

  • Gestión completa de usuarios
  • CRUD de canciones con búsqueda
  • Sistema de favoritos personalizado
  • Interfaz intuitiva y responsiva

Instalación y Uso

Guía rápida para poner en marcha el proyecto

Ejecución Rápida

Script automatizado para ejecutar todo el proyecto:

# Clonar repositorio git clone https://github.com/Juanrs69/lpa2-taller4.git cd lpa2-taller4 # Ejecutar tests y verificaciones ./run-tests.sh # Iniciar frontend cd frontend && pnpm dev

Configuración Manual

Pasos detallados para configuración completa:

# 1. Backend (API del lpa2-taller3) cd ../lpa2-taller3 python main.py # 2. Frontend (Nueva terminal) cd lpa2-taller4/frontend pnpm install pnpm dev # Frontend: http://localhost:3001 # API: http://127.0.0.1:8000
