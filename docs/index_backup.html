lpa2-taller4/ ├── frontend/ # Next.js App │ ├── app/ # App Router │ ├── components/ # React Components │ ├── hooks/ # Custom Hooks │ ├── lib/ # API Client │ └── __tests__/ # Frontend Tests ├── tests/ # Python Tests │ ├── test_documentacion.py # API Docs Tests │ ├── test_frontend_config.py # Config Tests │ ├── test_integracion.py # Integration Tests │ └── test_utils.py # Unit Tests ├── docs/ # GitHub Pages ├── documentacion_api_musica.json ├── api_endpoints_para_v0.json └── run-tests.sh # Test Runner

Instalación Rápida

1. Clonar repositorio

git clone https://github.com/Juanrs69/lpa2-taller4.git cd lpa2-taller4

2. Ejecutar tests

./run-tests.sh

3. Iniciar frontend

cd frontend && pnpm dev
